## Planned Course Library

The video course library will follow the STRATA Protocol's own five-stratum sequence, building from project classification through artifact preservation:

- **Stratum 1 — Classification:** How to classify a project correctly before any specification work begins, and why getting this right prevents architectural drift.
- **Stratum 2 — Derivation:** Walking through the full document hierarchy — Vision, BRD, PRD, Engineering Standards — in dependency order.
- **Stratum 3 — Authority Chain:** Setting up a constitutional mandate for your AI copilot — what it is, how to write it, and how it operates at session scope.
- **Stratum 4 — Execution Loop:** The human-gated cycle in practice — requirement analysis, phase planning, AI-assisted implementation, and human sign-off.
- **Stratum 5 — Artifact Trail:** Maintaining the living governance record across a full project lifecycle.
- **Workflow Series:** Applied sessions using GitHub Copilot, Cursor, and Claude in a fully governed STRATA environment.

See where each course sits in the development timeline on the [Roadmap](/changelog/roadmap).
